# Neurospark Corporation — Landing Page

**Kenya's Autonomous AI Company** | Nairobi, East Africa

A fully responsive, production-grade corporate landing page built with React + Vite.

## Tech Stack
- **React 18** + Vite 5
- **Lucide React** icons
- Inline CSS-in-JS (no Tailwind dependency)
- Deployed on **Vercel**

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```

## Deploy

This project is configured for one-click deployment on Vercel.
